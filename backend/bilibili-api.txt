### 视频播放相关

#### 直播流 (getRoomPlayInfo)
```
GET https://api.live.bilibili.com/xlive/web-room/v2/index/getRoomPlayInfo
```
| 参数 | 必填 | 说明 | 可选值 |
|------|------|------|-------|
| room_id | 是 | 直播间ID | - |
| protocol | 是 | 传输协议 | 0=HTTP, 1=HLS, 0,1=同时 |
| format | 是 | 容器格式 | 0=FLV, 1=TS, 2=FMP4, 0,1,2=全部 |
| codec | 是 | 视频编码 | 0=AVC(H.264), 1=HEVC(H.265), 2=AV1, 0,1,2=全部 |
| qn | 否 | 清晰度 | 80=流畅, 150=高清, 400=蓝光, 10000=原画, 20000=4K, 30000=杜比 |
| platform | 是 | 平台 | web/pc |
| ptype | 否 | 播放类型 | 8 |
| dolby | 否 | 杜比音效 | 5 |
| panorama | 否 | 全景视频 | 1 |
| only_audio | 否 | 仅音频 | 0/1 |
| web_location | 是 | 网页位置 | 444.8 |

**示例**
```
https://api.live.bilibili.com/xlive/web-room/v2/index/getRoomPlayInfo?room_id=1947277414&protocol=0,1&format=0,1,2&codec=0,1,2&qn=150&platform=web&ptype=8&dolby=5&panorama=1&web_location=444.8
```

#### 视频播放地址 (playurl)
```
GET https://api.bilibili.com/x/player/wbi/playurl
```
| 参数 | 必填 | 说明 | 可选值 |
|------|------|------|-------|
|avid|否|稿件avid|bvid和avid二选一|
|bvid|否|稿件bvid|avid和bvid二选一|
|cid|是|分P CID|-|
|qn|否|清晰度|16=360P, 32=480P, 64=720P, 80=1080P, 112=1080P+|
|fnval|否|视频格式|16=DASH, 64=HVC, 256=FLV, 512=FLV+AVC|
|fourk|否|4K|0/1|
|voice_balance|否|音质|0=标准, 1=低音, 2=高音|

```
| 参数 | 必填 | 说明 |
|------|------|------|
| like | 是 | 1=点赞, 2=取消 |
| csrf | 是 | CSRF Token |

#### 投币
```
POST https://api.bilibili.com/x/v2/view/coin/add
```
| 参数 | 必填 | 说明 |
|------|------|------|
| multiply | 是 | 投币数量 (1-2) |
| select_like | 是 | 是否同时点赞 (0/1) |
| csrf | 是 | CSRF Token |

#### 收藏
```
POST https://api.bilibili.com/x/v3/fav/resource/batch-deal

| rid | 是 | 稿件avid |

| add_media_ids | 否 | 要添加的收藏夹ID (逗号分隔) |
| del_media_ids | 否 | 要取消的收藏夹ID (逗号分隔) |
| csrf | 是 | CSRF Token |
